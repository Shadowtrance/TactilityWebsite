# M5Stack Tab5

Both hardware variants of the Tab5 are supported.

## Features

- ✅ SD card
- ✅ Wi-Fi via ESP32-C6
- ✅ Sound (I2S)
- ✅ IO Expanders (both)
- ✅ BMI270
