# M5Stack PaperS3

The touch screen isn't super responsive currently. 

When turning off the device, the Tactility launcher remains visible. The beeper will beep to confirm power off.

## Features

- ✅ SD card
- ✅ BMI270 IMU
- ✅ BM8563 RTC
- ✅ Power/Battery status
- ✅ USB Mass Storage
